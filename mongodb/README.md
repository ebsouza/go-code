## MongoDB connection

Setup
```
go mod init github.com/ebsouza/go-code/mongodb

go get go.mongodb.org/mongo-driver/mongo@v1.15.0

go get github.com/joho/godotenv@v1.5.1
```

Run
```
go run main.go
```
